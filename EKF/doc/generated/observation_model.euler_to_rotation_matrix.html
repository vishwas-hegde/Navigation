observation_model.euler_to_rotation_matrix¶

observation_model.euler_to_rotation_matrix(roll: float, pitch: float, yaw: float) → ndarray¶

Create the overall 3D rotation matrix from a roll-pitch-yaw sequence.

This function generates a 3D rotation matrix by composing individual rotation matrices for roll (around the x-axis), pitch (around the y-axis), and yaw (around the z-axis).

Parameters:

  • roll (float) – The roll angle in radians.

  • pitch (float) – The pitch angle in radians.

  • yaw (float) – The yaw angle in radians.

Returns:

The overall 3D rotation matrix.

Return type:

numpy.ndarray

Example: To create a rotation matrix from a roll-pitch-yaw sequence (30 degrees roll, 45 degrees pitch, 60 degrees yaw), use:

rotation_matrix = euler_to_rotation_matrix(np.pi/6, np.pi/4, np.pi/3)
