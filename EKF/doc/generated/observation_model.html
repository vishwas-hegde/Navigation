observation_model¶

This is a toolkit library for use as the observation model for the Nonlinear Kalman Filter and Particle Filter projects.

Functions

estimate_pose(data)

Estimates the pose (rotation and translation) from given image and object points using solvePnP.

euler_from_rvec(rvec)

Calculates the Euler angles from the rotation vector using OpenCV's Rodrigues method to calculate the rotation matrix

euler_to_rotation_matrix(roll, pitch, yaw)

Create the overall 3D rotation matrix from a roll-pitch-yaw sequence.

process_data(datafile)

Process a datastream provided in a .mat file, estimating positions and rotations.

rotation_matrix_to_euler(rotation_matrix)

Convert a 3D rotation matrix to Euler angles using the roll-pitch-yaw sequence.

rotation_matrix_x(theta)

Create a 3D rotation matrix around the x-axis (roll).

rotation_matrix_y(theta)

Create a 3D rotation matrix around the y-axis (pitch).

rotation_matrix_z(theta)

Create a 3D rotation matrix around the z-axis (yaw).

wrap_to_2pi(angle)

Wrap an angle to the range [0, 2*pi).

wrap_to_pi(angle)

Wrap an angle to the range [-pi, pi).
