observation_model.estimate_pose¶

observation_model.estimate_pose(data: dict) → tuple¶

Estimates the pose (rotation and translation) from given image and object points using solvePnP. This is an example solution for Task #1 in the Nonlinear Kalman Filter Project.

Parameters:

data (dict) –

A dictionary containing the necessary data for pose estimation. It should have the following keys:

  • ’p1’, ‘p2’, ‘p3’, ‘p4’: Image points (2D) as numpy arrays.

  • ’id’: Object point IDs as a numpy array.

Returns:

A tuple of rotation and translation vectors if the pose estimation is valid, otherwise (None, None).

Return type:

tuple

Note:

The solvePnP function requires the following additional variables to be defined:

  • camera_matrix: The camera intrinsic matrix.

  • dist_coeffs: The camera distortion coefficients.

  • rotation: Initial estimate of the rotation vector.

  • translation: Initial estimate of the translation vector.
