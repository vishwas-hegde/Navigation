observation_model.rotation_matrix_to_euler¶

observation_model.rotation_matrix_to_euler(rotation_matrix: ndarray) → tuple¶

Convert a 3D rotation matrix to Euler angles using the roll-pitch-yaw sequence.

This function takes a 3x3 rotation matrix and decomposes it into Euler angles in radians using the roll-pitch-yaw (RPY) sequence. The RPY sequence represents rotations around the coordinate axes, with roll (rotation around the X-axis), pitch (rotation around the Y-axis), and yaw (rotation around the Z-axis).

Parameters:

rotation_matrix (numpy.ndarray): The 3x3 rotation matrix representing the orientation.

Returns:

tuple: A tuple containing the roll, pitch, and yaw angles in radians.

Example: To convert a rotation matrix to Euler angles, use:

rotation_matrix = np.array([[0.866, -0.5, 0.0],
                           [0.5, 0.866, 0.0],
                           [0.0, 0.0, 1.0]])
roll, pitch, yaw = rotation_matrix_to_euler(rotation_matrix)
