observation_model.process_data¶

observation_model.process_data(datafile)¶

Process a datastream provided in a .mat file, estimating positions and rotations.

This function loads a .mat data file specified by datafile and processes the datastream within it. For each data entry in the stream, it estimates the position and rotation and collects the timestamps.

Parameters:

datafile (str) – The filename of the .mat data file to process.

Returns:

A tuple containing three numpy arrays:

  • positions (numpy.ndarray): An array of estimated 3D positions.

  • rotations (numpy.ndarray): An array of estimated 3D rotations.

  • times (numpy.ndarray): An array of timestamps corresponding to the data entries.

Return type:

tuple

Example: To process a datafile named ‘sample_data.mat’, use:

>>> positions, rotations, times = process_data('sample_data.mat')
