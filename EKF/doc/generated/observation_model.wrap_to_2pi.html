observation_model.wrap_to_2pi¶

observation_model.wrap_to_2pi(angle)¶

Wrap an angle to the range [0, 2*pi).

This function takes an angle in radians and wraps it to the range [0, 2*pi). It ensures that the resulting angle is within this range by performing a modulo operation with 2*pi.

Parameters:

angle (float) – The input angle in radians.

Returns:

The wrapped angle in the range [0, 2*pi).

Return type:

float
